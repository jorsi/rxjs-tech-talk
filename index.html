What Is RxJS?

RxJS is a library for composing asynchronous and event-based programs by using observable sequences. It provides one core type, the Observable, satellite types (Observer, Schedulers, Subjects) and operators inspired by Array#extras (map, filter, reduce, every, etc) to allow handling asynchronous events as collections.

What does that even mean?

Basically, Observables are event emitters that can be subscribed to by multiple Observers. They let us compose our user interfaces with small, independent components that can each subscribe to one Observable emitting data.
